## 什么是WiFi通话？

WiFi通话是一项让你的手机通过WiFi网络连接（而非蜂窝网络）拨打和接听常规电话的功能。对方不需要WiFi通话功能或任何特殊APP。对他们来说，这就是一个来自你手机号码的普通电话。

与微信语音、WhatsApp或Skype等网络电话APP不同，WiFi通话使用的是你运营商的电话系统。来电显示的是你的真实号码，可以拨打普通电话号码（包括座机），并使用你现有套餐的通话时长和功能。

## WiFi通话的工作原理

### 技术流程

当WiFi通话开启且手机检测到WiFi连接时，语音通话可以通过网络而非蜂窝基站传输。通话从你的手机经WiFi到达运营商服务器，然后进入传统电话网络到达接收方。

切换是无缝的。如果你在WiFi通话过程中走出WiFi覆盖范围，很多手机会在通话中切换到蜂窝网络而不掉线。反向切换也可以——从蜂窝切到WiFi信号更强时。

### 语音质量

WiFi通话通常使用高清语音（HD Voice），提供比传统电话更清晰的音频。由于WiFi连接通常有比蜂窝信号更大的带宽，WiFi通话的质量实际上可能优于蜂窝通话，尤其是在室内手机信号弱的时候。

### 数据使用量

WiFi通话每分钟大约使用1MB流量。在大多数家庭和企业网络连接上，这完全可以忽略不计。

## WiFi通话为什么重要

### 室内覆盖问题

蜂窝信号在穿过建筑材料时会减弱。混凝土、金属甚至节能窗户都能大幅降低信号强度。如果你在家里或办公室经常掉线，而室外覆盖还好，就是室内信号穿透不佳导致的。

WiFi通话通过完全绕过蜂窝基站来解决这个问题。只要你有可用的WiFi连接，通话就会清晰稳定。

### 农村和偏远地区

蜂窝覆盖有限但有卫星、固定无线或DSL宽带的区域可以利用WiFi通话来维持电话服务。这对有宽带但远离基站的农村住户特别有意义。

### 地下室和地下空间

位于地下室的办公室、公寓和工作空间通常完全没有蜂窝信号。WiFi通话在这些死角提供完整的电话功能。

### 国际使用

这是WiFi通话最具价值的场景。在海外连接WiFi后，WiFi通话让你使用美国号码拨打和接听电话，不产生漫游费。通话通过网络路由到你的运营商，被视为国内通话。

对旅行者来说，这意味着从酒店WiFi、机场网络或咖啡馆的网络免费打电话回美国。不需要漫游套餐，不用按分钟付费。对于经常需要和美国这边联系的华人来说，这个功能特别实用——在中国连上WiFi就能用美国号码打电话，对方看到的就是你的美国号码。

## WiFi通话与国际旅行

### 省钱效果

国际漫游通话费在很多套餐上高达每分钟$1至$3。一个30分钟的电话可能要花$30至$90。同样的电话通过WiFi打则不额外收费。

两周的旅行中每天打电话回家，节省的费用轻松超过$200。

### 运营商的限制

并非所有运营商都支持在海外使用WiFi通话，即使国内支持。部分运营商主动限制国际WiFi通话，或标记在国外大量使用该功能的账户。

这种限制一方面是出于反欺诈考虑，另一方面是因为运营商更希望你购买国际漫游套餐而不是通过WiFi通话免费使用。

### 支持国际WiFi通话的运营商

旅行前确认你的运营商和套餐是否支持在美国以外使用WiFi通话。查看细则，因为有些套餐技术上支持该功能，但对长期海外使用设有限制。

## 如何开启WiFi通话

### iPhone上

1. 打开**设置**
2. 点击**电话**
3. 点击**WiFi通话**
4. 开启**在此iPhone上启用WiFi通话**
5. 提示时确认紧急地址

### 安卓（三星）

1. 打开**设置**
2. 点击**连接**
3. 点击**WiFi通话**
4. 开启开关

### 安卓（Pixel）

1. 打开**设置**
2. 点击**网络和互联网**
3. 点击**SIM卡**并选择你的运营商
4. 开启**WiFi通话**

注意：WiFi通话需要手机和运营商都支持。如果你在设置中找不到该选项，你的运营商可能在当前套餐上不支持此功能。

## WiFi通话常见问题排解

### 电话仍然走蜂窝网络

如果手机有强蜂窝信号，可能会优先使用蜂窝。你可以在设置中将WiFi设为首选通话方式，寻找"WiFi优先"或"蜂窝优先"选项。

### WiFi通话质量差

WiFi通话质量取决于你的网络连接。如果出现断断续续的音频或掉线，检查网速——WiFi通话需要至少1 Mbps的上下行速度。同时检查其他设备是否占用了太多WiFi带宽。

### WiFi通话不可用

部分旧手机不支持WiFi通话。部分运营商将其限制在特定设备或套餐类型上。如果设置中没有出现开关，联系运营商确认兼容性。

### 紧急电话

WiFi通话支持911紧急电话，但定位精度可能不如蜂窝。开启WiFi通话时，运营商会要求你注册一个地址用于紧急服务。搬家后记得更新。

## WiFi通话 vs 网络电话APP

| 功能 | WiFi通话 | 网络电话APP（微信、WhatsApp等） |
|---|---|---|
| 使用你的真实号码 | 是 | 否（使用APP ID） |
| 拨打座机 | 是 | 通常需额外付费 |
| 对方需要安装APP | 否 | 是 |
| 使用运营商套餐 | 是 | 独立 |
| 紧急电话 | 是 | 否 |
| 通话质量 | 高清语音 | 不一定 |

WiFi通话和网络电话APP并非替代关系，而是服务于不同的目的。WiFi通话是通过WiFi扩展你的常规电话服务，而网络电话APP提供独立的通信渠道。
